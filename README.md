### Como utilizar

```shell

  # Primeiro execute.
    yarn
  # ou
    yarn add

```

```shell

  # Assim que as dependências forem baixadas execute.
    yarn server
  # para iniciar a api

```

```shell

  # logo após execute.
    yarn dev
  # para iniciar a aplicação

```

```shell

  # Para testes.
    yarn test

```
